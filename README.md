### Usage

```
npm install relit
```

### Contributing

#### Quick Start
```
git clone git@github.com:francoislaberge/relit.git
cd reflash
npm install
npm start
```

#### Publishing to NPM

```
npm version patch|minor|major
npm publish
git push && git push --tags
```

### Usage

```
npm install relit
```

### Contributing

#### Quick Start
```
git clone git@github.com:francoislaberge/relit.git
cd relit
npm install
npm start
```

#### Other Commands

 - `npm run clean`
 - `npm run build`
 - `npm run watch`

#### Publishing to NPM

```
npm version patch|minor|major
npm publish
git push && git push --tags
```
